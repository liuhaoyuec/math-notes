# 随机信号的测度：功率谱密度 (PSD) 与正交分解

## 1. 核心代数定理：为什么 $S_X(f) = S_S(f) + S_N(f)$ 严格成立？
在时域中，接收信号 $X(t)$ 是有用信号 $S(t)$ 与噪声 $N(t)$ 的线性叠加：
$$X(t) = S(t) + N(t)$$

要证明它们在频域的功率可以**直接代数相加**，必须通过自相关函数 $R_X(\tau)$ 进行严谨的空间内积展开。

**第一步：展开自相关函数（内积的线性分配律）**
$$R_X(\tau) = E[X(t) X(t+\tau)] = E[(S(t) + N(t))(S(t+\tau) + N(t+\tau))]$$
将其完全展开为四项的期望：
$$R_X(\tau) = E[S(t)S(t+\tau)] + E[N(t)N(t+\tau)] + E[S(t)N(t+\tau)] + E[N(t)S(t+\tau)]$$

**第二步：利用“统计独立”消除正交交叉项**
在通信系统的物理现实中，发送的信号 $S(t)$ 与大自然的背景噪声 $N(t)$ 是**绝对统计独立**的，且通常噪声的均值为零（$E[N(t)] = 0$）。
根据概率论，两个独立随机变量乘积的期望等于期望的乘积：
$$E[S(t)N(t+\tau)] = E[S(t)] \cdot E[N(t+\tau)] = E[S(t)] \cdot 0 = 0$$
同理，$E[N(t)S(t+\tau)] = 0$。

这在泛函空间中意味着：**信号向量与噪声向量是绝对正交的，它们的内积（交叉项）严格为 0。**
因此，式子瞬间坍缩为：
$$R_X(\tau) = R_S(\tau) + R_N(\tau)$$

**第三步：傅里叶变换的线性映射**
对等式两边同时做傅里叶变换。由于傅里叶变换是线性算子，和的变换等于变换的和：
$$\mathscr{F}\{R_X(\tau)\} = \mathscr{F}\{R_S(\tau)\} + \mathscr{F}\{R_N(\tau)\}$$
**绝对结论：**
$$S_X(f) = S_S(f) + S_N(f)$$
**代数本质：** 这就是希尔伯特空间中的**勾股定理**。正交基底上的能量可以直接标量相加。

## 2. 严密推导：功率谱密度究竟是怎么算出来的？
面对无限长且不可预测的随机过程 $X(t)$，我们不能直接做傅里叶变换（因为 $\int_{-\infty}^{\infty} |X(t)| dt = \infty$，不满足狄利克雷绝对可积条件）。
数学家通过**极限与测度**的思想，给出了 PSD 的严谨推导。

**步骤 1：时域截断（制造有限能量信号）**
既然无限长不能算，我们就先截取 $[-T/2, T/2]$ 这一段，强行构造一个截断信号 $X_T(t)$。
因为它是有限的，所以它存在绝对合法的傅里叶变换 $X_T(f)$：
$$X_T(f) = \int_{-T/2}^{T/2} X_T(t) e^{-j2\pi ft} dt$$

**步骤 2：利用帕塞瓦尔定理计算平均功率**
根据帕塞瓦尔定理 (Parseval's Theorem)，频域能量等于时域能量。我们在区间 $T$ 内求平均功率：
$$P_T = \frac{1}{T} \int_{-T/2}^{T/2} |X_T(t)|^2 dt = \int_{-\infty}^{\infty} \frac{|X_T(f)|^2}{T} df$$
此时，被积函数 $\frac{|X_T(f)|^2}{T}$ 物理意义极其明确：它是**截断信号在频域上的功率分布密度**。

**步骤 3：取统计平均与极限（定义出绝对的 PSD）**
这里有两个致命问题：
1. $X(t)$ 是随机的，所以每次截取算出来的 $\frac{|X_T(f)|^2}{T}$ 都在剧烈抖动。为了干掉随机性，必须取**数学期望 $E[\cdot]$**。
2. 截断会带来误差。为了还原真实的无限长信号，必须让观测时间 **$T \to \infty$**。

**终极严谨定义：** 随机过程 $X(t)$ 的功率谱密度 $S_X(f)$ 被严格定义为：
$$S_X(f) = \lim_{T \to \infty} E\left[ \frac{|X_T(f)|^2}{T} \right]$$

---

## 3. 工程落地的终极桥梁：维纳-辛钦定理
上述严密定义在数学上无懈可击，但在工程上根本没法算（因为你不可能真的让 $T \to \infty$ 去做积分运算）。



伟大的**维纳-辛钦定理 (Wiener-Khinchin Theorem)** 用极其复杂的实变函数积分证明了：
只要 $X(t)$ 是广义平稳随机过程（WSS），上述那个带着极限和期望的变态公式，**在数学上严格等价于它的自相关函数的傅里叶变换！**
$$S_X(f) = \int_{-\infty}^{\infty} R_X(\tau) e^{-j2\pi f \tau} d\tau$$

**底层逻辑闭环：**
我们彻底绕开了直接对随机信号算傅里叶变换的数学死局。我们先在时域求出确定的统计度量 $R_X(\tau)$，再对这个确定的函数做傅里叶变换，完美得到了随机过程在频域的绝对能量分布 $S_X(f)$。
