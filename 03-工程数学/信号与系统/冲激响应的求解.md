冲激响应的求解

**算式列写与实战求解框架**
冲激响应 $h(t)$ 是系统在激励为 $\delta(t)$ 时的[[零状态响应]]。考场上求解 $n$ 阶常系数微分方程的 $h(t)$，最严谨且不易出错的方法是**奇异函数匹配法结合辅助响应法**。

**通用模型**：$y^{(n)}(t) + a_{n-1}y^{(n-1)}(t) + \dots + a_0 y(t) = b_m f^{(m)}(t) + \dots + b_0 f(t)$

**计算步骤（考场标准流程）**：
1. **构造辅助方程求 $h_1(t)$**：
   * 剥离右侧导数项，先求极简方程的响应 $h_1(t)$：
     $h_1^{(n)}(t) + a_{n-1}h_1^{(n-1)}(t) + \dots + a_0 h_1(t) = \delta(t)$
   * **考场直接背诵的初始条件**：在 $t=0_+$ 瞬间，仅最高阶导数跃变为 1，其余全为 0：
     $h_1(0_+) = h_1'(0_+) = \dots = h_1^{(n-2)}(0_+) = 0$
     $h_1^{(n-1)}(0_+) = 1$
   * 结合特征根求出 $t>0$ 时的齐次解，代入上述初始条件定系数，得到 $h_1(t) = [ \sum C_k e^{\lambda_k t} ] \cdot u(t)$。

2. **线性组合求极终 $h(t)$**：
   * 根据原方程右侧的系数和导数阶次，对 $h_1(t)$ 进行线性组合与求导：
     $h(t) = b_m h_1^{(m)}(t) + b_{m-1} h_1^{(m-1)}(t) + \dots + b_0 h_1(t)$
   * **考场易错点**：对 $h_1(t)$ 求导时，**千万不要漏掉 $u(t)$ 的求导！** 由乘积法则，导数中必然会产生包含 $\delta(t)$ 及其导数的项（当 $m \ge n$ 时）。
