# 正交函数集 (Orthogonal Function Sets)

## 1. 核心定义
在区间 $(t_1, t_2)$ 内，如果两个实变量函数 $\phi_1(t)$ 和 $\phi_2(t)$ 满足其内积为 0，即：
$$\int_{t_1}^{t_2} \phi_1(t)\phi_2(t) dt = 0$$
则称这两个函数在该区间内是**相互正交**的。

## 2. 什么是正交函数集？
如果有一个函数集合 $\{ \phi_1(t), \phi_2(t), ..., \phi_n(t) \}$，其中**任意两个不同的函数**在区间 $(t_1, t_2)$ 内都满足正交条件，且每个函数自身的能量不为 0（即 $\int \phi_i^2(t)dt = K_i \neq 0$），这就构成了一个正交函数集。

## 3. 物理与工程意义
正交性是整个 [[傅里叶级数]] 和频域分析的灵魂所在。
它意味着信号可以像在三维空间中被分解为互相垂直的 x、y、z 坐标轴一样，被无损地分解为一系列互不干扰的正交基函数（例如不同频率的正弦波、余弦波或虚指数信号）。在这些“轴”上，信号的能量是线性叠加的，不会发生耦合。
