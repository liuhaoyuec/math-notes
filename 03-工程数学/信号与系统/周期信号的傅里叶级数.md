# 周期信号的傅里叶级数 (FS)

## 1. 两种核心形式
任何满足狄里赫利条件的周期信号 $f(t)$（周期为 $T$，基频 $\omega_0 = \frac{2\pi}{T}$）都可以展开为正交函数集的线性组合：

**① 三角形式 (直观物理意义)**
$$f(t) = a_0 + \sum_{n=1}^{\infty} [a_n \cos(n\omega_0 t) + b_n \sin(n\omega_0 t)]$$
或者化为幅度-相位形式：$f(t) = c_0 + \sum_{n=1}^{\infty} c_n \cos(n\omega_0 t + \theta_n)$
- $c_0 = a_0$ 为直流分量。
- $c_n$ 为第 $n$ 次谐波的振幅。

**② 指数形式 (后续推导 FT 的数学基石)**
利用欧拉公式，可以将三角形式转化为复指数形式：
$$f(t) = \sum_{n=-\infty}^{\infty} F_n e^{jn\omega_0 t}$$
- $F_n$ 是复数，称为**复傅里叶系数**，包含了幅度和相位信息。$F_n = \frac{1}{T} \int_{-T/2}^{T/2} f(t) e^{-jn\omega_0 t} dt$。

## 2. 周期信号频谱的三大特点 (高频考点)
1. **离散性**：频谱不是连续的曲线，而是只在基频 $\omega_0$ 的整数倍（$0, \pm\omega_0, \pm2\omega_0...$）处有非零值。这就好比梳子的齿。
2. **谐波性**：所有谱线的频率都是基频的整数倍。
3. **收敛性**：随着谐波次数 $n$ 的增大，频谱的幅度 $|F_n|$ 总体趋势是逐渐衰减趋于零的（高频能量越来越少）。
