# 知识点 (1)：拉普拉斯变换与收敛域判断方法

## 1. 核心定义
拉普拉斯变换是解决不满足绝对可积条件信号的终极武器，引入了衰减因子 $e^{-\sigma t}$，构成复频率 $s = \sigma + j\omega$。

* **双边拉普拉斯变换：**
  $$X(s) = \int_{-\infty}^{\infty} x(t) e^{-st} dt$$

* **单边拉普拉斯变换（自带初始状态，工程最常用）：**
  $$X(s) = \int_{0^{-}}^{\infty} x(t) e^{-st} dt$$

> [!warning] 教练批注
> 注意积分下限是 $0^{-}$！这决定了它能处理包含冲激函数 $\delta(t)$ 的初始状态。做复频域计算题时，谁把 $0^{-}$ 丢了，谁就直接挂科！

## 2. 致命考点：收敛域（ROC）
没有收敛域，拉普拉斯变换就是一堆废纸！同一个表达式对应因果还是反因果信号，全靠 ROC 决定。

**收敛域的判断法则（必须形成肌肉记忆）：**
1. **有限长信号：** 收敛域为整个 $s$ 平面（极少数可能除去 $s=0$ 或 $s=\infty$）。
2. **右边信号（如因果信号 $t > 0$）：** 收敛域在最右边极点的右侧，即 $Re(s) > \sigma_{max}$。
3. **左边信号（如反因果信号 $t < 0$）：** 收敛域在最左边极点的左侧，即 $Re(s) < \sigma_{min}$。
4. **双边信号：** 收敛域是两个极点之间的带状区域，即 $\sigma_1 < Re(s) < \sigma_2$。如果不相交，则根本不存在拉普拉斯变换！
