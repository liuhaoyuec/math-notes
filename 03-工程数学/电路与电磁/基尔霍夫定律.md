# 基尔霍夫定律 (Kirchhoff's Laws)

## 1. 基尔霍夫电流定律 (KCL)
**定义**：在集总电路中，任意时刻，对任意节点，流出（或流入）该节点的所有支路电流的代数和为零。
$$\sum_{k=1}^{n} i_k(t) = 0$$
**场论本质**：KCL 是 [[电流与电流密度]] 中**电荷守恒定律**在集总电路中的宏观体现。由于节点不能积累电荷（连续性方程 $\nabla \cdot \mathbf{J} = 0$），所以流入的总电荷率必须等于流出的总电荷率。

## 2. 基尔霍夫电压定律 (KVL)
**定义**：在集总电路中，任意时刻，沿任意闭合回路，所有支路电压的代数和为零。
$$\sum_{k=1}^{m} u_k(t) = 0$$
**场论本质**：KVL 是**电场力做功与路径无关**这一性质（即电场的保守性）的体现。根据法拉第 [[电磁感应]] 定律 $\nabla \times \mathbf{E} = -\frac{\partial \mathbf{B}}{\partial t}$，在集总假设下，我们将随时间变化的磁通量 $\Phi_B$ 严密地限制在电感元件内部，因此在回路的其余空间中，电场可以视为保守场，闭合路径上的电势降落之和必然为零，即对应 [[电压、电势与电动势]] 的线积分性质。
