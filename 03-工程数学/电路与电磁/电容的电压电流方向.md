# 电容的电压电流方向 (Capacitor VCR)

## 1. 推荐选取策略
通常选取**关联参考方向**。这样符合“电流流入正极板，极板电荷量增加，电压升高”的物理直觉。

## 2. 微分方程
假设电容值为 $C$：
- **当 $u, i$ 为关联参考方向时**：
  $$i = C \frac{du}{dt}$$
- **当 $u, i$ 为非关联参考方向时**：
  $$i = -C \frac{du}{dt}$$
  *(注：如果电流从 $-$ 极流入，说明正极板的电荷在流失，电压 $u$ 的变化率必然为负，所以公式前必须加负号配平)*
