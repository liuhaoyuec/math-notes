# 理想无源元件 (Ideal Passive Elements)

**重要前提**：以下所有的微积分方程（伏安特性关系 VCR），均建立在电压 $u$ 和电流 $i$ 采用**关联参考方向**（电流从电压 $+$ 极流入）的基础之上。若为非关联方向，等号右侧需加负号。

## 1. 电阻 (Resistor, R)
**物理本质**：消耗电磁能量（转化为热能）的理想模型。
**代数方程 (欧姆定律)**：
$$u(t) = R \cdot i(t)$$
或用电导 $G = 1/R$ 表示：$i(t) = G \cdot u(t)$

## 2. 电容 (Capacitor, C)
**物理本质**：储存 [[电场]] 能量的理想模型。其电荷量 $q$ 与电压 $u$ 成正比：$q(t) = C \cdot u(t)$。
**微分方程**：电流是电荷对时间的导数（$i = \frac{dq}{dt}$），因此：
$$i(t) = C \frac{du(t)}{dt}$$
**积分方程**：
$$u(t) = \frac{1}{C} \int_{-\infty}^{t} i(\tau) d\tau = u(t_0) + \frac{1}{C} \int_{t_0}^{t} i(\tau) d\tau$$
*(注意：这表明电容电压具有记忆性，且在电流有限的情况下，电容电压不能发生突变。)*

## 3. 电感 (Inductor, L)
**物理本质**：储存 [[磁场]] 能量的理想模型。根据 [[电磁感应]]，其磁链 $\psi$ 与电流 $i$ 成正比：$\psi(t) = L \cdot i(t)$。
**微分方程**：根据法拉第电磁感应定律（$u = \frac{d\psi}{dt}$），因此：
$$u(t) = L \frac{di(t)}{dt}$$
**积分方程**：
$$i(t) = \frac{1}{L} \int_{-\infty}^{t} u(\tau) d\tau = i(t_0) + \frac{1}{L} \int_{t_0}^{t} u(\tau) d\tau$$
*(注意：电感电流具有记忆性，且在电压有限的情况下，电感电流不能发生突变。)*
