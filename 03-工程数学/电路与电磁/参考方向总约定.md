# 参考方向总约定 (Sign Conventions)

## 1. 关联参考方向 (Associated Reference Direction)
**定义**：对于某个元件，人为假定的电流 $i$ 的箭头，**指向**人为假定的电压 $u$ 的 **$+$** 极。（即电流顺着电压降落的方向流）。
**适用场景**：强烈建议对所有**无源元件（电阻、电容、电感）**默认采用此方向。
**功率意义**：在关联方向下，若算出的功率 $P = ui > 0$，表示该元件在**吸收/消耗**功率（符合无源元件本性）。

## 2. 非关联参考方向 (Non-associated Reference Direction)
**定义**：人为假定的电流 $i$ 的箭头，**指向**人为假定的电压 $u$ 的 **$-$** 极。（即电流逆着电压降落的方向流）。
**适用场景**：通常用于**有源元件（电压源、电流源）**，或者在复杂拓扑图中被迫形成此方向。
**功率意义**：在非关联方向下，若算出的功率 $P = ui > 0$，表示该元件在**发出/提供**功率（符合电源本性）。
