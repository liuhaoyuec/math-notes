# 数字信号处理 (DSP) 面试高频考点速查表

## 1. 奈奎斯特采样定理 (Nyquist Sampling Theorem)
**核心定义**：将模拟信号转换为数字信号时，采样频率 $f_s$ 必须大于等于信号最高频率分量 $f_m$ 的两倍，即 $f_s \ge 2f_m$，才能无失真地恢复原信号。

**疑惑点突破**：
* **什么是混叠 (Aliasing)？** 当 $f_s < 2f_m$ 时，高频频谱会搬移并重叠到低频频段，导致原始信号无法恢复。
* **工程应用**：实际系统中，在ADC（模数转换器）之前，必须加一个**抗混叠滤波器**（低通滤波器），滤除高于 $f_s/2$ 的高频噪声，强制满足采样定理。

## 2. Z变换与傅里叶变换的关系
**核心定义**：Z变换是离散时间傅里叶变换 (DTFT) 的推广。DTFT 是在单位圆上的 Z变换。

**疑惑点突破**：
* **为什么有了 DTFT 还要引入 Z变换？** 很多离散信号（如阶跃信号）不满足绝对可积条件，DTFT 不收敛。引入复变量 $z = r e^{j\omega}$ 中的衰减因子 $r$，使得原本不收敛的信号能在特定的收敛域 (ROC) 内存在 Z变换，从而方便系统分析。
* **系统稳定性**：因果系统稳定的充要条件是系统函数 $H(z)$ 的所有极点都必须位于 Z平面的**单位圆内**。

## 3. DFT 与 FFT 的本质区别
**核心概念**：
* **DFT (离散傅里叶变换)**：在时域和频域都是离散的、有限长的。它是计算机唯一能够直接处理的傅里叶变换形式。
* **FFT (快速傅里叶变换)**：**不是一种新的变换**。它是为了高效计算 DFT 而提出的一种**算法**。

**疑惑点突破**：
* **FFT 快在哪里？** 利用了旋转因子 $W_N^{kn} = e^{-j\frac{2\pi}{N}kn}$ 的**周期性**和**对称性**，将长序列的 DFT 分解为短序列的 DFT，大大减少了复数乘法和加法的运算量。复杂度从 $O(N^2)$ 降到了 $O(N \log_2 N)$。

## 4. FIR 与 IIR 数字滤波器的核心对比

| 对比维度 | IIR (无限长单位冲激响应) | FIR (有限长单位冲激响应) |
| :--- | :--- | :--- |
| **结构特点** | 有反馈回路（递归结构） | 无反馈回路（非递归结构） |
| **相位特性** | 非线性相位 | **可以做到严格的线性相位** |
| **稳定性** | 存在极点出单位圆的风险，需校验 | 永远稳定（极点全部在原点） |
| **运算效率** | 达到相同滤波效果，阶数低，计算量小 | 达到相同滤波效果，阶数高，计算量大 |
| **适用场景** | 语音通信等对相位要求不高的场景 | 图像处理、数据传输等要求波形不失真的场景 |

**疑惑点突破**：
* **什么是线性相位？为什么重要？** 线性相位意味着信号所有频率成分在通过滤波器时，产生的**时间延迟是完全相同的**，组合起来后的输出信号**波形不会发生失真**。
