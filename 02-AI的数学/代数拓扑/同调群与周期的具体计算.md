# 同调群与周期的具体计算

这是椭圆积分映射到 $\mathbb{C}/\Lambda$ 的核心计算机制。如何算出那个“格” $\Lambda$？

## 1. 拓扑结构与同调基
复曲线 $E(\mathbb{C})$ 是一个亏格 $g=1$ 的紧致黎曼面（环面）。
其第一同调群 $H_1(E(\mathbb{C}), \mathbb{Z}) \cong \mathbb{Z} \oplus \mathbb{Z}$。
这意味着环面上存在两个基本的、不可收缩的独立闭合回路（生成元），记为 $\alpha$ 和 $\beta$。

## 2. 周期的严谨定义
将全纯微分 $\omega = \frac{dx}{y}$ 沿着同调基 $\alpha$ 和 $\beta$ 进行闭合积分，得到的值称为**基本周期 (Fundamental Periods)**：
$$\omega_1 = \oint_\alpha \frac{dx}{y}, \quad \omega_2 = \oint_\beta \frac{dx}{y}$$
由这两个复数生成的自由阿贝尔群 $\Lambda = \mathbb{Z}\omega_1 \oplus \mathbb{Z}\omega_2$ 就是周期格。

## 3. 具体计算方法 (分支割线法)

在实际计算中，$\omega_1$ 和 $\omega_2$ 如何求出数值？
方程 $y^2 = (x-e_1)(x-e_2)(x-e_3)$ 在复平面上有三个分支点 $e_1, e_2, e_3$ 和无穷远点 $\infty$。
实用的计算转换为在复平面上带割线的实积分（假设 $e_1, e_2, e_3$ 为实数且 $e_1 < e_2 < e_3$）：
1. **实周期 $\omega_1$** (对应两根之间的积分)：
   $$\omega_1 = 2 \int_{e_1}^{e_2} \frac{dx}{\sqrt{(x-e_1)(x-e_2)(x-e_3)}}$$
2. **纯虚周期 $\omega_2$**：
   $$\omega_2 = 2 \int_{e_2}^{e_3} \frac{dx}{\sqrt{(x-e_1)(x-e_2)(x-e_3)}}$$
*(注：上述积分可通过算术-几何平均 (AGM) 算法或超几何函数进行高精度数值计算。不能用初等函数直接写出原函数。)*

## 4. Abel-Jacobi 映射的计算闭环
对于曲线上任意点 $P=(x_0, y_0)$，积分 $\int_O^P \frac{dx}{y}$ 的值 $z$，无论路径如何选择，其差值必为 $m\omega_1 + n\omega_2$。
因此，严格的态射表达式为：
$$P \mapsto \left( \int_O^P \frac{dx}{y} \right) \pmod{\Lambda}$$
