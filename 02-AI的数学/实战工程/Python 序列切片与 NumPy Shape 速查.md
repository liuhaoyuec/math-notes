# Python 序列切片与 NumPy Shape 速查

## 1. 类型与表示

| 类型 | 表示 | 可变 | 切片结果 |
| :--- | :--- | :--- | :--- |
| 列表 | $[a_0,\dots,a_{n-1}]$ | 是 | 列表 |
| 元组 | $(a_0,\dots,a_{n-1})$ | 否 | 元组 |

- 单元素元组：$(a,)$ 或 $a,$  
  $(a)$ 是带括号表达式，非元组。

---

## 2. 索引

- 正索引：$i\in[0,n-1]$，$S[i]=a_i$
- 负索引：$i\in[-n,-1]$，$S[i]=S[n+i]$

---

## 3. 切片 $S[\text{start}:\text{stop}:\text{step}]$

### 3.1 规则

- **左闭右开**：包含 $\text{start}$，不包含 $\text{stop}$
- **默认值**：

| | $\text{step}>0$ | $\text{step}<0$ |
| :--- | :--- | :--- |
| $\text{start}$ | $0$ | $-1$（即 $n-1$） |
| $\text{stop}$ | $n$ | $-n-1$（即 $-1$） |

- **边界裁剪**：超界自动缩至有效范围，不报错
- **空结果条件**：
  - $\text{step}>0$ 且 $\text{start} \ge \text{stop}$
  - $\text{step}<0$ 且 $\text{start} \le \text{stop}$

### 3.2 索引序列生成

$$i_0 = \text{start},\quad i_{k+1}=i_k+\text{step}$$

终止：
- $\text{step}>0$ 时 $i_k<\text{stop}$
- $\text{step}<0$ 时 $i_k>\text{stop}$

只保留 $0\le i_k<n$ 的元素。

## 4. 常用写法速查

设 $S=(2,3,5,7,11,13)$

| 写法 | 结果 | 说明 |
| :--- | :--- | :--- |
| $S[:3]$ | $(2,3,5)$ | 前 $3$ 项 |
| $S[::2]$ | $(2,5,11)$ | 步长 $2$ |
| $S[::-1]$ | $(13,11,7,5,3,2)$ | 反转 |
| $S[:-2]$ | $(2,3,5,7)$ | 去后 $2$ 项 |
| $S[-2:]$ | $(11,13)$ | 取后 $2$ 项 |
| $S[4:1:-1]$ | $(11,7,5)$ | 索引 $4,3,2$ |
| $S[0:2:-1]$ | $()$ | 方向矛盾 |

---

## 5. NumPy `shape` 属性

- `arr.shape` 返回元组 $(d_0,\dots,d_{N-1})$
- $N=\text{len(shape)}$，元素总数 $=\prod d_i$

| 操作 | 含义 |
| :--- | :--- |
| $\text{shape}[k]$ | 第 $k$ 维长度 |
| $\text{shape}[-1]$ | 末维长度 |
| $\text{shape}[-2]$ | 倒数第二维长度 |
| $\text{shape}[-2:]$ | 末二维元组 |
| $\text{shape}[:-2]$ | 除末二维外元组 |

---

## 6. 边界情况

| `shape` | 维度数 | 说明 |
| :--- | :--- | :--- |
| $()$ | $0$ | 标量 |
| $(m,)$ | $1$ | 一维数组 |
| $(r,c)$ | $2$ | 二维数组 |
