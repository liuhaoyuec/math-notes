# 🧮 数学证明：Transformer 的“脸盲症”与位置编码的破局

> [!info] 核心定理
> 在不加位置编码时，纯粹的 Self-Attention 机制满足**排列等变性 (Permutation Equivariance)**。即：打乱输入的词序，输出的词序会跟着打乱，但每个词计算出的特征数值**绝对不变**。模型无法区分 `1+5` 和 `5+1`。

---

## 一、 证明：为什么原始 Transformer 是“脸盲”的？

假设输入序列为矩阵 $X \in \mathbb{R}^{N \times d}$ （$N$ 为序列长度，$d$ 为特征维度）。
注意力机制的简化核心公式为：
$$Y = \text{softmax}\left( \frac{X W_Q (X W_K)^T}{\sqrt{d_k}} \right) X W_V$$

**【证明过程】**

**1. 引入置换矩阵 (Permutation Matrix)**
定义一个 $N \times N$ 的置换矩阵 $P$。它的作用是交换矩阵的行（即打乱输入序列的字序）。
根据线性代数性质，置换矩阵是一个正交矩阵，满足极其重要的一点：
$$P^T P = I \quad \text{(单位矩阵)}$$

**2. 输入打乱后的序列**
我们将打乱词序后的新输入记为 $X' = PX$。我们来看它经过各个权重矩阵后的变化：
$$Q' = X' W_Q = P X W_Q = P Q$$
$$K' = X' W_K = P X W_K = P K$$
$$V' = X' W_V = P X W_V = P V$$
*(物理意义：字与字依然没有耦合，只是各自在矩阵里的行号改变了)*

**3. 计算打乱后的注意力得分矩阵 $S'$**
$$S' = Q' (K')^T$$
代入上一步的结果，并利用转置运算法则 $(AB)^T = B^T A^T$：
$$S' = (PQ) (PK)^T = P Q K^T P^T$$
因为原得分矩阵 $S = Q K^T$，所以：
$$S' = P S P^T$$
*(数学意义：这是一个相似变换，得分矩阵的行和列被以同一种方式同步打乱了)*

**4. 计算最终输出 $Y'$**
对 $S'$ 按行进行 Softmax 归一化，得到 $A' = P A P^T$。最后乘以 $V'$：
$$Y' = A' V' = (P A P^T) (P V)$$
利用矩阵乘法的结合律：
$$Y' = P A (P^T P) V$$
代入正交性质 $P^T P = I$：
$$Y' = P A I V = P (A V)$$
因为原来的输出 $Y = AV$，所以得出终极结论：
$$Y' = P Y$$

> [!danger] 结论证明完毕
> $Y' = PY$ 意味着：输入如果是乱序的 $X$，输出就是同样乱序的 $Y$。词语在空间中“随波逐流”，但它吸收到的上下文信息总量（也就是那 $d$ 个浮点数）**完全没有发生任何改变**！这就是 Transformer 丧失顺序感知能力的数学铁证。

## 二、 证明：位置编码是如何打破这一定理的？

为了打破 $Y' = PY$ 的僵局，我们引入固定的位置编码矩阵 $E \in \mathbb{R}^{N \times d}$，其第 $i$ 行代表“第 $i$ 号座位的绝对坐标”。

**【破局推导】**

**1. 宏观视角的打破**
加上位置编码后，原本的输入变成了：
$$X_{\text{pos}} = X + E$$
此时，如果我们用置换矩阵 $P$ 试图打乱词序，**座位号 $E$ 是固定在空间中不动的**！因此打乱后的新输入是：
$$X'_{\text{pos}} = P X + E$$
注意这里的数学断言：
$$P X + E \neq P(X + E)$$
既然等式左边不成立，那么后续的一系列推导将全面崩溃，$P^T P = I$ 的消元魔法再也无法触发，排列等变性被彻底打破。
