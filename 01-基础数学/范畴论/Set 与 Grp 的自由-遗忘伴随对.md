# Set 与 Grp 的自由-遗忘伴随对

这是最经典的代数构造之一，连接了无结构的集合与具有单一二元运算的群。

##  functors (函子对)
- **右伴随 (遗忘函子)** $U: \mathbf{Grp} \to \mathbf{Set}$
  - **作用**：将群 $G$ 的乘法结构“遗忘”，退化为纯粹的底层集合。
- **左伴随 (自由函子)** $F: \mathbf{Set} \to \mathbf{Grp}$
  - **作用**：给定任意集合 $X$（字母表），构造由 $X$ 生成的**自由群 (Free Group)** $F(X)$。其元素是 $X$ 中元素及其逆元构成的所有有限“既约字”，运算为字的拼接。

## 伴随同构 (Adjunction Isomorphism)
$$Hom_{\mathbf{Grp}}(F(X), G) \cong Hom_{\mathbf{Set}}(X, U(G))$$

**直观理解**：
要在自由群 $F(X)$ 和任意群 $G$ 之间建立群同态，你**只需要（且必须）**把集合 $X$ 中的每个“字母”映射到 $G$ 中的某个元素。一旦这步完成，剩下的运算结构会自动由自由群的泛性质（Universal Property）补全。

## 复合关系
- 向上复合：通过遗忘函子可以走向更底层的 [[Set 范畴]]。
- 向下复合：与 [[Grp与Ab的伴随对]] 复合，可以得到集合到阿贝尔群的伴随。
