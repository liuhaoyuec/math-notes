# 椭圆曲线上 Abel-Jacobi 映射的严格积分计算

**标签：** #椭圆积分 #局部坐标 #拉回 (Pullback) #流形计算 #射影几何

## 1. 核心任务设定
**目标：** 在由 $y^2 = 4x^3 - Ax - B$ 定义的复椭圆曲线 $E$ 上，计算积分 $z = \int_O^P \frac{dx}{y}$。
**起点 $O$：** 射影平面上的无穷远点 $[X:Y:Z] = [0:1:0]$。
**终点 $P$：** 仿射平面上的一个有限点 $(x_P, y_P)$。
**几何对象：** 1-形式 $\omega = \frac{dx}{y}$。

由于起点 $O$ 和终点 $P$ 不能放在同一个常规欧氏坐标系里计算（无穷大不能直接积分），我们必须**严格按照流形的定义，将路径切分，并在两个重叠的坐标卡中分别使用转移映射拉回计算**。

## 2. 第一段积分：在坐标卡 $U_Y$ 中（处理无穷远起点 $O$）

**选取坐标卡：** 起点 $O = [0:1:0]$ 满足 $Y \neq 0$，所以进入坐标卡 $U_Y$。
**定义局部坐标：** $t = \frac{X}{Y}, s = \frac{Z}{Y}$。此时 $O$ 点变为纯欧氏空间的原点 $(t,s) = (0,0)$。
**曲线的局部方程：** 将 $x = t/s, y = 1/s$ 代入原方程，整理得：
$$s = 4t^3 - Ats^2 - Bs^3$$

**转移映射与微分拉回：**
我们要把常规空间的 $\omega = \frac{dx}{y}$ 转移到当前的 $(t,s)$ 坐标系中。
通过全微分法则：
$$x = \frac{t}{s} \implies dx = \frac{s dt - t ds}{s^2}$$
$$y = \frac{1}{s}$$
将它们代入 $\omega$ 进行拉回 (Pullback)：
$$\omega = \frac{dx}{y} = s \left( \frac{s dt - t ds}{s^2} \right) = dt - \frac{t}{s}ds$$

**消去 $ds$ 转化为纯一元积分：**
对曲线局部方程 $s = 4t^3 - Ats^2 - Bs^3$ 两边同时求全微分：
$$ds = 12t^2 dt - A(s^2 dt + 2ts ds) - 3Bs^2 ds$$
提取 $ds$：
$$ds = \frac{12t^2 - As^2}{1 + 2Ats + 3Bs^2} dt$$
将 $ds$ 代回 $\omega = dt - \frac{t}{s}ds$，我们就得到了一个完全由 $t$ 表达的标准形式 $\omega = f(t)dt$。
由于 $s \approx 4t^3$（当 $t \to 0$ 时），函数 $f(t)$ 在原点 $t=0$ 处是**全纯的（没有分母为零的奇点）**。

**执行计算：** 选定路径 $\gamma_1$，从起点 $O$ ($t=0$) 积分到一个中间点 $Q$ ($t=t_Q$)：
$$I_1 = \int_0^{t_Q} \left( 1 - \frac{t}{s(t)} \cdot \frac{12t^2 - As(t)^2}{1 + 2Ats(t) + 3Bs(t)^2} \right) dt$$
（这完全是一个普通的欧氏定积分，可以直接用数值方法或级数展开算出一个确切的复数）。

## 3. 第二段积分：在坐标卡 $U_Z$ 中（处理常规仿射空间）

**坐标切换：** 当我们到达中间点 $Q$ 时，$Z \neq 0$，我们顺利进入了你最熟悉的仿射坐标卡 $U_Z$。
**局部坐标：** $x = \frac{X}{Z}, y = \frac{Y}{Z}$。
在 $U_Z$ 中，曲线方程就是 $y^2 = 4x^3 - Ax - B$。

**转移映射：** 
点 $Q$ 的坐标从 $U_Y$ 切换到 $U_Z$：$x_Q = \frac{t_Q}{s_Q}$。
微分形式不需要拉回，因为它本来就是穿的 $(x,y)$ 的衣服：$\omega = \frac{dx}{y}$。

**执行计算：** 选定路径 $\gamma_2$，从中间点 $Q$ ($x=x_Q$) 积分到终点 $P$ ($x=x_P$)：
$$I_2 = \int_{x_Q}^{x_P} \frac{dx}{\pm\sqrt{4x^3 - Ax - B}}$$
（这也是一个标准的黎曼积分）。

## 4. 最终结果汇总 (流形积分的一致性)

根据流形上的积分定义，这条跨越了“无穷远”和“有限远”的路径总积分，就是两个坐标卡中纯欧氏积分的直接相加：
$$z = \int_O^P \frac{dx}{y} = I_1 + I_2$$

**结论：** 
整个计算过程严格贯彻了**“局部同胚到欧氏空间 $\to$ 利用转移映射拉回微分形式 $\to$ 在普通欧氏拓扑中算定积分”**的现代几何计算框架。在 $U_Y$ 里的复杂代换，本质上就是高维版本的“换元积分法”，它吃掉了把无穷远拉伸到原点所产生的拓扑形变。
