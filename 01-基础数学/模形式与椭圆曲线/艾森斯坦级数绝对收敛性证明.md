# 艾森斯坦级数绝对收敛性证明 (为什么要求 $k > 2$)

**目标**：证明当 $k > 2$ 时，级数 $\sum_{(m,n) \neq (0,0)} \frac{1}{|m\tau + n|^k}$ 收敛。

### 1. 将二维求和指标“分层” (按无穷范数分类)
为了把二维的双重求和变成一维求和，我们按照点到原点的“逻辑距离”对网格点 $(m,n)$ 进行分层。
定义层级 $N = \max(|m|, |n|)$。
由于 $(m,n) \neq (0,0)$，所以 $N$ 取正整数 $N = 1, 2, 3, \dots$。

### 2. 计算第 $N$ 层的点数
当 $\max(|m|, |n|) = N$ 时，这些格点在复平面上恰好落在一个中心在原点、边长为 $2N$ 的正方形的**边界**上。
这个边界上的整数点数量可以通过计算大正方形减去内部小正方形的整数点得到：
$$\text{第 } N \text{ 层的点数} = (2N + 1)^2 - (2N - 1)^2 = 8N$$

### 3. 建立分母 $|m\tau + n|$ 的下界 (核心代数操作)
我们需要说明，在第 $N$ 层上，所有的格点距离原点 $|m\tau + n|$ 都有一个统一的下界。
考虑复平面上的连续函数 $f(x, y) = |x\tau + y|$。
因为 $\tau$ 在上半平面（$\text{Im}(\tau) > 0$），所以 $\tau$ 和 $1$ 在实数域上线性无关。因此，只有当 $x=0, y=0$ 时，$f(x,y)$ 才为 $0$。

在集合 $\{(x,y) \in \mathbb{R}^2 \mid \max(|x|, |y|) = 1\}$（这是一个闭正方形边界，属于紧集）上，$f(x,y)$ 必然能取到一个大于 $0$ 的最小值，记为 $C$（$C > 0$ 且 $C$ 只依赖于 $\tau$）。

对于任意第 $N$ 层的点 $(m,n)$，提取因子 $N$：
$$|m\tau + n| = N \left| \frac{m}{N}\tau + \frac{n}{N} \right|$$
因为 $\max\left(\left|\frac{m}{N}\right|, \left|\frac{n}{N}\right|\right) = 1$，所以括号里的部分必定大于等于 $C$。
得出严格下界：
$$|m\tau + n| \ge C \cdot N$$

### 4. 放缩与转化为 $p$-级数
现在把上面两步的结果代入原始级数进行放缩。我们将双重求和按照层级 $N$ 重新组合：
$$\sum_{(m,n) \neq (0,0)} \frac{1}{|m\tau + n|^k} = \sum_{N=1}^{\infty} \left( \sum_{\max(|m|,|n|)=N} \frac{1}{|m\tau + n|^k} \right)$$

对于第 $N$ 层的每一个项，分母都被放缩为最小可能值 $(CN)^k$；同时这一层总共有 $8N$ 个项。所以这一层的和满足：
$$\sum_{\max(|m|,|n|)=N} \frac{1}{|m\tau + n|^k} \le \frac{8N}{(CN)^k} = \frac{8}{C^k} \cdot \frac{1}{N^{k-1}}$$

将所有层加起来：
$$\sum_{(m,n) \neq (0,0)} \frac{1}{|m\tau + n|^k} \le \frac{8}{C^k} \sum_{N=1}^{\infty} \frac{1}{N^{k-1}}$$

### 5. 结论
右边的级数 $\sum_{N=1}^{\infty} \frac{1}{N^{k-1}}$ 是标准的一维 $p$-级数。
根据微积分的基础定理，**当且仅当指数 $k - 1 > 1$，即 $k > 2$ 时，该级数收敛**。
根据比较判别法，原二维级数绝对收敛。
