# 模形式理论中的傅里叶展开 ($q$-展开)

> [!abstract] 核心认知
> 在模形式理论中，由于复变函数特有的“全纯性”，傅里叶展开彻底摆脱了实分析（或信号处理）中的“逐点收敛”、“吉布斯现象”等病态问题。这里的级数是**绝对且一致收敛**的，可以直接画等号，并允许无条件地进行逐项求导和积分。

## 1. 展开的严谨前提条件
对于定义在复上半平面 $\mathbb{H} = \{ \tau \in \mathbb{C} \mid \text{Im}(\tau) > 0 \}$ 上的函数 $f(\tau)$，只需满足以下**两个条件**，即可进行完美的傅里叶展开：

* **条件一：平移周期性**
    对于任意 $\tau \in \mathbb{H}$，满足 $f(\tau+1) = f(\tau)$。
    *(在模形式中，这直接由弱模性公式代入平移矩阵 $T$ 得到。)*
* **条件二：上半平面全纯性 (Holomorphy)**
    $f(\tau)$ 在上半平面 $\mathbb{H}$ 上处处全纯（复可导且无奇点）。
    *(全纯性保证了函数不仅连续，而且具有无限次可导的极度平滑性，这是级数完美一致收敛的根本数学保障。)*

---

## 2. 完整的展开公式
满足上述条件的复变函数 $f(\tau)$，必然可以唯一地展开为如下级数形式：

**形式一：经典指数形式**
$$f(\tau) = \sum_{n=-\infty}^{\infty} a_n e^{2\pi i n \tau}$$

**形式二：$q$-展开形式（模形式标准写法）**
引入映射 $q = e^{2\pi i \tau}$，直接将指数替换，得到关于 $q$ 的洛朗级数：
$$f(\tau) = \sum_{n=-\infty}^{\infty} a_n q^n$$

---

## 3. 傅里叶系数 $a_n$ 的计算公式
系数 $a_n$ 通过沿上半平面水平线段的积分求得。
设 $\tau = x + iy$。固定任意一个虚部 $y > 0$（即在 $\mathbb{H}$ 中画一条高度为 $y$ 的水平线），在一个周期 $x \in [0, 1]$ 上积分：

$$a_n = \int_{0}^{1} f(x+iy) e^{-2\pi i n (x+iy)} dx$$

> [!info] 积分与路径高度无关
> 由于 $f(\tau)$ 是全纯的，根据柯西积分定理，上述积分算出的常数 $a_n$ **与你选择的 $y$ 的具体数值完全无关**。无论在多高的地方积分，算出的频域系数都是同一个。

---

## 4. 模形式的最终约束（无穷远全纯）
上述公式是上半平面所有全纯周期函数的通用展开。
当我们额外要求 $f(\tau)$ 是一个**模形式**时，它必须满足第三个条件：“在无穷远处 ($i\infty$) 全纯”。
这在代数上的直接强制后果就是**截断所有负次幂**：

$$\text{若 } f \text{ 在 } i\infty \text{ 处全纯，则必须有：} \quad a_n = 0 \quad (\forall n < 0)$$

因此，模形式最终的 $q$-展开公式严格定型为只有非负次幂的泰勒级数：
$$f(\tau) = \sum_{n=0}^{\infty} a_n q^n = a_0 + a_1 q + a_2 q^2 + \dots$$
