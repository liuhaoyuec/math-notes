# 余切函数极点展开式证明 (Mittag-Leffler 展开)

**目标**：证明 $f(z) = \pi \cot(\pi z)$ 与 $g(z) = \frac{1}{z} + \sum_{n=1}^{\infty} \left( \frac{1}{z - n} + \frac{1}{z + n} \right)$ 完全相等。

> [!warning] 疑惑点指出：关于级数的收敛
> 注意右边 $g(z)$ 的写法。它**不能**直接写成无序的 $\sum_{n \neq 0} \frac{1}{z-n}$，因为那样是不绝对收敛的。把它写成 $\left( \frac{1}{z-n} + \frac{1}{z+n} \right) = \frac{2z}{z^2 - n^2}$ 后，级数就变成了 $\sum \frac{2z}{z^2-n^2}$，这就像 $\sum \frac{1}{n^2}$ 一样，保证了在去掉整数点的复平面上是**绝对且紧致一致收敛**的。

## 1. 分析奇点与留数 (Poles and Residues)
首先，观察 $f(z) = \pi \frac{\cos(\pi z)}{\sin(\pi z)}$。
它的奇点出现在分母为 0 的地方，即 $\sin(\pi z) = 0$，解得 $z = n \in \mathbb{Z}$（所有整数点）。
计算在任意整数点 $z=n$ 处的留数（Residue）：
$$\text{Res}(f, n) = \lim_{z \to n} (z-n) \pi \frac{\cos(\pi z)}{\sin(\pi z)} = \lim_{z \to n} \pi \frac{\cos(\pi z)}{\pi \cos(\pi z)} = 1$$
*(这里用了洛必达法则对分母求导)*。
结论：$f(z)$ 在所有整数点有简单极点，且留数全为 $1$。

再看 $g(z)$，极其明显，它在所有 $z = n \in \mathbb{Z}$ 处也有简单极点，且由定义可知，留数同样全为 $1$。

## 2. 构造差值函数，消除奇点
我们定义一个差值函数：
$$h(z) = f(z) - g(z)$$
因为 $f(z)$ 和 $g(z)$ 在所有整数点 $n$ 处的极点类型和留数**完全一样**，它们相减时，发散的那一部分（主部）刚好精确抵消。
在复分析中，如果一个奇点的主部被消除了，这个奇点就变成了**可去奇点 (Removable singularity)**。
结论：$h(z)$ 在整个复平面上没有任何极点，它是一个**整函数 (Entire function)**。

## 3. 证明有界性与刘维尔定理
为了对付整函数，我们构造一族避开整数极点的正方形路径 $C_N$。正方形的四个顶点在 $(N + \frac{1}{2})(\pm 1 \pm i)$ 处（即卡在整数的中间）。
1. 当 $N \to \infty$ 时，在这个正方形的边界上，通过纯代数放缩可以严格证明 $|\pi \cot(\pi z)|$ 总是小于某个绝对常数 $M_1$（因为它在水平方向收敛到 $\pm \pi i$，在垂直方向周期性有界）。
2. 同理，$g(z)$ 在这族正方形边界上也是一致有界的，绝对值小于 $M_2$。

因此，$h(z) = f(z) - g(z)$ 在无穷扩大的正方形边界上有界。由最大模原理，整函数在内部的模长不可能超过边界。
结论：$h(z)$ 是一个**在整个复平面上有界的整函数**。

此时，复分析中最强悍的判据之一，**刘维尔定理 (Liouville's Theorem)** 出场了：
*定理：任何有界的整函数，必定是常数。*
所以，$h(z) \equiv C$。即 $f(z) - g(z) = C$。

## 4. 利用奇偶性确定常数
我们来考察 $f(z)$ 和 $g(z)$ 的奇偶性。
* $f(-z) = \pi \cot(-\pi z) = -\pi \cot(\pi z) = -f(z)$。所以 $f(z)$ 是**奇函数**。
* $g(-z) = \frac{1}{-z} + \sum_{n=1}^{\infty} \left( \frac{1}{-z - n} + \frac{1}{-z + n} \right) = -g(z)$。所以 $g(z)$ 也是**奇函数**。

既然两者都是奇函数，它们的差 $h(z)$ 必然也是奇函数：
$$h(-z) = -h(z)$$
但我们前面已经用刘维尔定理证明了 $h(z)$ 是一个常数 $C$。
代入奇偶性方程：$C = -C$，解得：
$$C = 0$$

## 5. 最终结论
因为 $h(z) \equiv 0$，所以：
$$f(z) = g(z)$$
即 $\pi \cot(\pi z) = \frac{1}{z} + \sum_{n=1}^{\infty} \left( \frac{1}{z - n} + \frac{1}{z + n} \right)$ 严格成立。证明完毕。
