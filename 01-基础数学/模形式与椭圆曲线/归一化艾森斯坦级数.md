# 归一化艾森斯坦级数 $E_k(\tau)$ 与伯努利数

> [!abstract] 核心动机
> 原始的艾森斯坦级数 $G_k(\tau)$ 带有超越数（如 $\pi$）。为了将其用于深刻的数论研究（如证明全等式或构造 L 函数），我们必须将其常数项强制化为 $1$，从而使其所有傅里叶系数都变成纯粹的**有理数**（甚至整数）。

## 1. 原始展开式回顾
我们已知权为偶数 $k \ge 4$ 的艾森斯坦级数展开式为：
$$G_k(\tau) = 2\zeta(k) + \frac{2 \cdot (2\pi i)^k}{(k-1)!} \sum_{n=1}^{\infty} \sigma_{k-1}(n) q^n$$
其中 $\sigma_{k-1}(n)$ 是除数函数。

## 2. 引入欧拉公式与伯努利数 $B_k$
为了处理常数项 $2\zeta(k)$，我们引入欧拉对偶数阶黎曼 $\zeta$ 函数的惊人恒等式：
$$2\zeta(k) = - \frac{(2\pi i)^k B_k}{k!}$$
这里的 $B_k$ 是**伯努利数**，它们是一组完全由有理数构成的序列，定义自生成函数 $\frac{x}{e^x - 1} = \sum_{k=0}^{\infty} \frac{B_k}{k!} x^k$。
前几个偶数阶伯努利数为：
* $B_2 = \frac{1}{6}$
* $B_4 = -\frac{1}{30}$
* $B_6 = \frac{1}{42}$
* $B_8 = -\frac{1}{30}$
* $B_{10} = \frac{5}{66}$
* $B_{12} = -\frac{691}{2730}$

## 3. 实施归一化 (Normalization)
我们将 $G_k(\tau)$ 整体除以它的常数项 $2\zeta(k)$（即除以 $-\frac{(2\pi i)^k B_k}{k!}$），定义出**归一化艾森斯坦级数 $E_k(\tau)$**：
$$E_k(\tau) = \frac{G_k(\tau)}{2\zeta(k)}$$

代入化简后，超越数 $(2\pi i)^k$ 被完美约掉，阶乘项化简为 $\frac{k!}{(k-1)!} = k$。我们得到了极其清爽的算术形态：
$$E_k(\tau) = 1 - \frac{2k}{B_k} \sum_{n=1}^{\infty} \sigma_{k-1}(n) q^n$$

---

## 4. 两个最基础的模形式基石：$E_4$ 与 $E_6$
代入具体的 $k=4$ 和 $k=6$：

**权为 4 的归一化艾森斯坦级数 $E_4$**：
已知 $B_4 = -1/30$，系数为 $-\frac{2 \times 4}{-1/30} = 240$。
$$E_4(\tau) = 1 + 240 \sum_{n=1}^{\infty} \sigma_3(n) q^n = 1 + 240q + 2160q^2 + \dots$$

**权为 6 的归一化艾森斯坦级数 $E_6$**：
已知 $B_6 = 1/42$，系数为 $-\frac{2 \times 6}{1/42} = -504$。
$$E_6(\tau) = 1 - 504 \sum_{n=1}^{\infty} \sigma_5(n) q^n = 1 - 504q - 16632q^2 + \dots$$

> [!success] 极其重要的结论
> $E_4$ 和 $E_6$ 不仅在无穷远 $i\infty$ 处全纯（因为没有 $q$ 的负次幂），而且它们的常数项均为 $1$，且所有的内部傅里叶系数都是**整数**。它们将成为构建整个模形式空间的“原点基底”。
