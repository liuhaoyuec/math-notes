# 泰勒展开 (Taylor Series)

**严谨定义**：
设复变函数 $f(z)$ 在以 $z_0$ 为中心的**开圆盘** (Open Disk) $D = \{z \in \mathbb{C} \mid |z - z_0| < R\}$ 内是[[全纯函数|全纯（解析）]]的。那么在该圆盘内，$f(z)$ 可以唯一地展开为幂级数：
$$f(z) = \sum_{n=0}^{\infty} c_n (z - z_0)^n$$
其中系数由公式给出：
$$c_n = \frac{f^{(n)}(z_0)}{n!} = \frac{1}{2\pi i} \oint_\gamma \frac{f(\zeta)}{(\zeta - z_0)^{n+1}} d\zeta$$
（$\gamma$ 是 $D$ 内任意一条包围 $z_0$ 的简单闭曲线）。

> [!note]- 核心证明思路 (柯西积分公式 + 几何级数)
> 1. 由柯西积分公式，对圆盘内任意一点 $z$：$f(z) = \frac{1}{2\pi i} \oint_\gamma \frac{f(\zeta)}{\zeta - z} d\zeta$
> 2. 将分母巧妙变形提取 $(\zeta - z_0)$：
>    $\frac{1}{\zeta - z} = \frac{1}{(\zeta - z_0) - (z - z_0)} = \frac{1}{\zeta - z_0} \cdot \frac{1}{1 - \frac{z - z_0}{\zeta - z_0}}$
> 3. 因为 $z$ 在 $\gamma$ 内部，所以 $\left| \frac{z - z_0}{\zeta - z_0} \right| < 1$。
> 4. 利用等比数列求和公式（几何级数）展开：$\frac{1}{1-q} = \sum_{n=0}^{\infty} q^n$。
> 5. 代入积分号并交换积分与求和（由一致收敛性保证），即得泰勒系数公式。

### 💡 泰勒展开的“取巧”计算方法

在实际计算中，**永远不要傻乎乎地去求 $n$ 阶导数**（除非是极其罕见的怪函数）。我们总是通过已知标准展开式进行**代数变形**：

**1. 牢记五大基本展开 (背诵级)：**
* $\frac{1}{1-z} = 1 + z + z^2 + z^3 + \dots \quad (|z| < 1)$  *(万物之源)*
* $e^z = 1 + z + \frac{z^2}{2!} + \frac{z^3}{3!} + \dots \quad (|z| < \infty)$
* $\sin z = z - \frac{z^3}{3!} + \frac{z^5}{5!} - \dots$
* $\cos z = 1 - \frac{z^2}{2!} + \frac{z^4}{4!} - \dots$
* $\ln(1+z) = z - \frac{z^2}{2} + \frac{z^3}{3} - \dots \quad (|z| < 1)$

**2. 变量代换法 (Substitution)：**
求 $\frac{1}{1+z^2}$ 的展开？直接把 $-z^2$ 作为一个整体代入基本式 $\frac{1}{1-X}$ 中：
$\frac{1}{1 - (-z^2)} = 1 + (-z^2) + (-z^2)^2 + \dots = 1 - z^2 + z^4 - z^6 + \dots$

**3. 逐项求导/积分法 (Differentiation/Integration)：**
求 $\frac{1}{(1-z)^2}$ 的展开？
利用 $(\frac{1}{1-z})' = \frac{1}{(1-z)^2}$，直接对 $1+z+z^2+z^3+\dots$ 逐项求导：
$= 1 + 2z + 3z^2 + 4z^3 + \dots$ *(这正是我们在证明 $\wp(z)$ 展开时用的神技！)*

**4. 乘积与长除法 (待定系数法)：**
求 $\tan z = \frac{\sin z}{\cos z}$ 的前几项？
设 $\tan z = c_1 z + c_3 z^3 + \dots$
则 $(c_1 z + c_3 z^3)(1 - \frac{z^2}{2!} + \dots) = z - \frac{z^3}{3!} + \dots$，比较同次幂系数秒杀。
