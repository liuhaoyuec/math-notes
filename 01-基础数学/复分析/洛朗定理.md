# 洛朗定理的严密证明 (Laurent's Theorem)

**定理**：若 $f(z)$ 在圆环域 $D: r < |z - z_0| < R$ 内解析，则对于 $D$ 内的任意点 $z$，$f(z)$ 均可唯一展开为洛朗级数：
$$f(z) = \sum_{n=-\infty}^{\infty} c_n (z - z_0)^n$$

### 1. 构造双重边界的柯西积分
在圆环域 $D$ 内，取两个同心圆周 $C_1$ (半径为 $R_1$) 和 $C_2$ (半径为 $r_1$)，使得 $r < r_1 < |z - z_0| < R_1 < R$。
此时，点 $z$ 严格处于 $C_1$ 和 $C_2$ 围成的**闭合圆环**内部。
我们在 $C_1$ 和 $C_2$ 之间切一刀，将它们连成一条单连通的闭合路径。根据柯西积分公式，点 $z$ 处的函数值由边界上的积分决定。注意外圈 $C_1$ 是逆时针，内圈 $C_2$ 为了保持区域在左侧，必须是顺时针（等价于逆时针积分前加个负号）：
$$f(z) = \frac{1}{2\pi i} \oint_{C_1} \frac{f(\zeta)}{\zeta - z} d\zeta - \frac{1}{2\pi i} \oint_{C_2} \frac{f(\zeta)}{\zeta - z} d\zeta$$

这就是证明的物理起点：**外部边界的贡献 减去 内部边界的拉扯**。

### 2. 处理外圈积分 (产生正次幂 —— 泰勒部分)
对于 $C_1$ 上的点 $\zeta$，因为它在 $z$ 的外面，所以有 $|z - z_0| < |\zeta - z_0|$。
我们提取大项 $\zeta - z_0$，把积分核展开为几何级数：
$$\frac{1}{\zeta - z} = \frac{1}{(\zeta - z_0) - (z - z_0)} = \frac{1}{\zeta - z_0} \cdot \frac{1}{1 - \frac{z - z_0}{\zeta - z_0}}$$
因为公比 $\left| \frac{z - z_0}{\zeta - z_0} \right| < 1$，可以安全地展开为等比数列求和：
$$\frac{1}{\zeta - z} = \sum_{n=0}^{\infty} \frac{(z - z_0)^n}{(\zeta - z_0)^{n+1}}$$
代入外圈积分，交换积分与求和号：
$$\frac{1}{2\pi i} \oint_{C_1} \frac{f(\zeta)}{\zeta - z} d\zeta = \sum_{n=0}^{\infty} \left[ \frac{1}{2\pi i} \oint_{C_1} \frac{f(\zeta)}{(\zeta - z_0)^{n+1}} d\zeta \right] (z - z_0)^n$$
方括号里的系数，就是正次幂系数 $c_n$（$n \ge 0$）。

### 3. 处理内圈积分 (产生负次幂 —— 主部分)
这是最神奇的一步。对于 $C_2$ 上的点 $\zeta$，它在 $z$ 的里面，即 $|\zeta - z_0| < |z - z_0|$。
这次我们要提取的是 $z - z_0$，而且要注意前面有个负号：
$$-\frac{1}{\zeta - z} = \frac{1}{z - \zeta} = \frac{1}{(z - z_0) - (\zeta - z_0)} = \frac{1}{z - z_0} \cdot \frac{1}{1 - \frac{\zeta - z_0}{z - z_0}}$$
因为公比 $\left| \frac{\zeta - z_0}{z - z_0} \right| < 1$，再次展开为几何级数：
$$-\frac{1}{\zeta - z} = \sum_{m=0}^{\infty} \frac{(\zeta - z_0)^m}{(z - z_0)^{m+1}}$$
代入内圈积分：
$$-\frac{1}{2\pi i} \oint_{C_2} \frac{f(\zeta)}{\zeta - z} d\zeta = \sum_{m=0}^{\infty} \left[ \frac{1}{2\pi i} \oint_{C_2} f(\zeta) (\zeta - z_0)^m d\zeta \right] \frac{1}{(z - z_0)^{m+1}}$$
为了和前面的符号统一，令 $n = -(m+1)$。那么当 $m$ 从 $0$ 到 $\infty$ 时，$n$ 就从 $-1$ 到 $-\infty$。
括号里的积分化为 $\frac{1}{2\pi i} \oint_{C_2} \frac{f(\zeta)}{(\zeta - z_0)^{n+1}} d\zeta$，这就是负次幂系数 $c_n$（$n < 0$）。

### 4. 结论与组装
根据柯西积分定理，因为 $C_1$ 和 $C_2$ 之间的环带内 $f(z)$ 是解析的，所以计算系数 $c_n$ 时，积分路径无论取 $C_1$ 还是 $C_2$，或者圆环内的任何闭曲线 $\gamma$，结果都一样。
把正数次幂（来自外圈）和负数次幂（来自内圈反转）完美拼合：
$$f(z) = \sum_{n=-\infty}^{\infty} c_n (z - z_0)^n$$
其中 $c_n = \frac{1}{2\pi i} \oint_{\gamma} \frac{f(\zeta)}{(\zeta - z_0)^{n+1}} d\zeta$。证明完毕。
